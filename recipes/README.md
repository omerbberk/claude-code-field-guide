# Community Recipes 🧑‍🍳

Short, self-contained Claude Code workflows contributed by real users. One file = one recipe = one PR — this is the easiest place in the repo to make your first contribution.

## Add yours in 3 steps

1. Copy [`_template.md`](_template.md) to `recipes/your-recipe-name.md` (kebab-case).
2. Fill it in — keep the prompt exactly as you'd actually type it, with placeholders (`your-project`, `YOUR_API_KEY`) instead of anything real.
3. Open a PR. That's it. See [CONTRIBUTING.md](../CONTRIBUTING.md) for the ground rules (short version: no secrets, only things you've actually run).
